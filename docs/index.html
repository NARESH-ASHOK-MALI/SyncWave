How It Works

Engineered for perfect sync

A sophisticated audio pipeline built on Windows Core Audio (WASAPI).

1

System Audio Capture

WASAPI loopback captures all system audio in real-time at native quality — games, music, calls, everything.

2

Latency Measurement

Each device's hardware latency is measured via AudioClient.StreamLatency. The maximum is used as the sync baseline.

3

Delay Compensation

High-performance circular buffers insert calculated delays: maxLatency − deviceLatency + manualOffset

4

Volume Scaling & Output

Real-time volume scaling (0–200%) applied per-device, then routed through WasapiOut to each physical device.
